## 🚀 About This Repository

This repository documents my **hands-on cybersecurity learning journey** with a focus on **Security Operations Center (SOC) analysis**. It is a structured collection of my practical work, notes, and applied knowledge gained through real-world simulation labs.

The goal of this portfolio is to demonstrate not just learning, but the **ability to analyze, investigate, and document security events like a SOC Analyst**.
